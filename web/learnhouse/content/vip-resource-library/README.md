# VIP Resource Library — Structure

**Course slug:** `vip-resource-library`  
**Access:** `vip-members` group only (AE-VIP-MON / AE-VIP-YR)  
**Language:** English

Apprentice graduates on VIP also retain Apprentice course access per SKU matrix.

---

## Purpose

Ongoing reference materials for operators who maintain weekly review discipline. **Not** a live signal feed inside the LMS.

---

## Sections

### 1. SOP Archive

Static PDFs (mirror Apprentice worksheets + extensions):

### 2. Structured Trade Idea Format

Single guide page explaining how ideas are documented in VIP context:

- Context / higher-timeframe bias (education)
- Invalidation level
- Risk % and size reference
- Trigger description — **if/then**, not "enter now"
- Review questions after outcome

**Compliance line on page:**
> Examples illustrate process documentation. Not instructions to copy. You own every execution decision.

### 3. Automation Support Docs

- Tool-agnostic setup notes (education)
- Link to DWY service page for paid setup (`AE-DWY-001`)
- Kill-switch and demo-test checklist

**No** "passive income" framing.

### 4. Office Hours / Recordings (optional at launch)

- Unlisted replay links when sessions run
- Title format: `AE VIP Office YYYY-MM-DD` (neutral)
- Each recording opens with 15s compliance intro

---

## What NOT to put in VIP Library

- Guaranteed win screenshots as marketing
- "Copy this trade" alerts
- Credentials or broker login details
- P&L leaderboards

## LearnHouse setup

1. Create course **VIP Resource Library** (non-linear — resource sections OK)
2. Restrict enrollment to `vip-members` group
3. Do not auto-enroll Apprentice-only buyers
4. On VIP provision SOP: enroll **both** Apprentice + VIP Library
