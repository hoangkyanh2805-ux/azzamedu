## Swing or scalping - Big problem of trader

### What it is?

Jayce addresses the **identity crisis** killing traders: switching between **swing** (hold days, wider stop, fewer trades) and **scalping** (intraday, tight structure, more frequency) **weekly** without mastering either. Course taught **both** — Sep 26–29 scalps plus S03 swing POI theory — students must **pick primary mode** for next 90 days.

**Swing profile:**

- HTF H4/D1 POI, holds through sessions
- Lower frequency, larger structural stops
- Set & forget more common on power ≥20

**Scalping profile (Jayce day trading):**

- M5/M15 BOS bounce, US30/XAU focus
- Fixed or liquidity TP same day
- Sep 26 four-trade day model

**Big problem:** Scalp entry with swing stop size (or reverse) — **RR and psychology break**.

### How to define or apply it?

**90-day commitment test:**

1. Choose **scalp** OR **swing** primary.
2. Journal tag every trade `SCALP` or `SWING` — no hybrid tag unless defined rule (e.g., swing entry scalp partial).
3. Max trades/day: scalp 3–4; swing 1–2.
4. Review monthly — win rate and stress by mode.

**Jayce path:** Day trader scalper + **some** swing on gold H4 — but **sequenced**, not random.

**NCI Level 4** for day trading depth after Level 2 complete.

### Example

**Student A:** Swings EURUSD with M5 entries — stopped constantly — fix: align entry TF to TF/2 rule (S18) or switch to scalp pairs US30.

**Student B:** Scalps but holds losers overnight "swing" — fix: hard close or structural weekend rule.

## Tip 2025: Where should you start analyzing on clean chart?

### What it is?

**Clean chart** protocol — Jayce's **2025 tip** for students drowning in drawings: **start analysis fresh each session** in fixed order, no legacy rectangles.

**Start order (top-down):**

1. **D1** — bias one line, mark major liquidity
2. **H4** — structure BOS/CHoCH, premium/discount
3. **H1** — POI #1 candidates
4. **M15** — POI #2, FVG, session liquidity
5. **M5** — entry trigger only after above complete

**Clean chart means:**

- Remove yesterday's lines unless levels still **active**
- No indicators until structure drawn (optional NCI after)
- **Screenshot blank → finished markup** for journal

**Where NOT to start:** M1 random FVG, indicator arrow, Telegram level.

### How to define or apply it?

**Pre-session 10-minute routine:**

1. New chart layout or hide all drawings
2. Follow 5-step order — timer per step
3. If bias unclear at step 2 — **no step 5**
4. Save template in TradingView

**Link S18 TF/2:** Bias TF first, entry TF last.

### Example

**Monday morning:** Student starts M5 — sees FVG — long — stopped. Jayce replay: H4 premium short bias — FVG was **counter-trend noise**. Clean chart from D1 would skip.
