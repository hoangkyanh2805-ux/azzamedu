## Read all my strategy only through one day trading example + Scalping 5%/day 2700$

### What it is?

This opening **S12 journal lesson** is Jayce's **master template** — he claims you can understand **the entire SMC system** by reading **one fully documented trading day** end-to-end: pre-session bias, every markup, every skip, every entry, every loss cap, and end-of-day review. The Udemy title adds the **headline example**: **scalping 5%/day** and **$2700** — these are **historical journal numbers from a funded account day**, not a guarantee. They illustrate what **stacked A-grade scalps** at **1% risk × multiple R** can sum to when rules align.

**5%/day meaning in context:** Jayce uses **account %** the same way as Sep 26 (+5% net after −1.5% first trade). **5% in one day** is an **outlier highlight** for motivation — S16 monthly target lesson corrects expectation: aim for **consistent weekly process**, not daily 5% requirement.

**$2700 figure:** Dollar PnL on a **~$54k prop or personal account** at ~5% day — scales linearly with account size. Journal teaches **% and R first**; dollars second for emotional realism.

**What "read all strategy through one day" includes:**

- Double POI + BOS bounce entries (S03–S04)
- Signal power scoring per trade (S07)
- Weak vs strong case labeling (S09–S10)
- Loss cap and recovery without martingale (S08)
- Wyckoff cycle phase note on chart (S05)
- Lot size from calculator (S19)

**Deliverable:** PDF journal format from Resources tab — columns Jayce fills on camera.

### How to define or apply it?

**One-day journal template (copy to Notion/Excel):**

| Column | Content |
|--------|---------|
| Time | Session segment |
| Pair | US30, XAUUSD, etc. |
| HTF bias | D1/H4 one line |
| Setup type | Follow / break / scalp |
| POI #1 / #2 | Prices or zones |
| Power IB/IMB/Total | Numbers |
| Entry mode | S&F / CONF |
| Risk % | Per trade |
| Result R | Per trade |
| Result % acct | Per trade |
| Running daily % | Cumulative |
| Screenshot | Y/N |
| Notes | Footprint phase |

**Daily workflow:**

1. **Pre-market (15 min):** Mark 2–3 instruments; write **one bias sentence** each.
2. **Per setup:** Mark liquidity → wait sweep → score power → POI → trigger rule.
3. **Per fill:** Screenshot within 5 minutes; log before next scan.
4. **Post-session (20 min):** Sum R and %; tag best/worst **process** trade.
5. **Weekly:** Export; count mistake tags from S05/S06.

**5%/day scalp structure (example day Jayce walks):**

- Trade 1: +1.5% (1.5R)
- Trade 2: +2% (2R)
- Trade 3: +1.5% (1.5R)
- Trade 4: +0.5% breakeven management
- **Net ~+5%** — requires **no oversized loss** and **no rule breaks**

**Risk ceiling:** If Trade 1 is **−1.5%**, need **higher R** on winners or extra scalp — Jayce shows math on whiteboard.

### Example

**Annotated "2700$ day" walkthrough (process only)**

1. **06:00:** Jayce marks XAUUSD H4 bullish; US30 H4 bullish — **risk-on bias**.
2. **08:30:** US30 sweep long setup — double POI, power 20, BOS bounce — **+1.8% account**.
3. **10:00:** XAU pullback POI — confirmation, **+1.2%**.
4. **14:00:** US30 afternoon liquidity sweep short — premium POI, **+1.5%**.
5. **15:30:** Small scalp gold — **+0.7%**; stop trading.
6. **Total ~5.2%** ≈ **$2700** on stated account — journal PDF shows exact fills.

## My trading journal day - 03/10/2022 - Market cycle and Price action

### What it is?

**October 3, 2022** journal integrates **S04 market cycle theory** with **daily journal** — Jayce labels **accumulation / markup / distribution** phase on D1/H4 before taking intraday POIs. Title: **Market cycle and Price action** — trades filtered by **where in cycle** price sits, not indicator signals.

**Why cycle in journal:** Prevents buying **distribution** POIs labeled bullish because OB exists. Cycle phase = **context column** in journal.

### How to define or apply it?

**Journal column add:** `Cycle phase` — Acc / Markup / Dist / Markdown (or Wyckoff A–E sketch).

**Rules:**

- **Follow-trend POIs** preferred in markup/markdown legs.
- **Break-trend** (S03 Case 1–3) only with CHoCH + liquidity sweep at cycle extreme.
- If cycle **unclear** — journal **no trade** hours.

**Price action notes:** Wick behavior at POI, speed of displacement, session highs — qualitative bullets beside scores.

### Example

**03/10/2022 replay**

- D1 XAUUSD: Jayce marks **distribution** into highs — bias cautious long on intraday only.
- US30: **Markup** leg — follow-trend long POIs favored.
- Journal shows **skipped** gold long at premium — cycle filter.
- Took US30 pullback double POI in markup — **+2.5R**.
- End note: "Cycle wrong = OB trap."
