## M3L1 — Daily SOP checklist

### Objective
Install a repeatable pre-session, in-session, and post-session routine.

### Video
`[PENDING: YouTube unlisted URL]` · Title: `AE Apprentice M3L1`

### Key points

- **Pre-session (15–20 min):** desk readiness, economic calendar scan, mark levels, write plan — no orders yet.
- **In-session:** execute only plans that passed the risk gate; no new ideas mid-tilt.
- **Post-session:** log trades, screenshot chart with annotations, note emotional vs structured tags.
- Same checklist every day reduces decision fatigue — the SOP carries you when motivation drops.
- Demo and live use the **same** SOP — only size changes when you deliberately choose.
- Print the Daily SOP Checklist; tape it to your monitor for week 1.

## M3L2 — Weekly review template

### Objective
Run a structured weekly review focused on habits and process — not P&L marketing.

### Video
`[PENDING: YouTube unlisted URL]` · Title: `AE Apprentice M3L2`

### Key points

- Weekly review answers: Did I follow SOP? Where did emotion override structure? What one habit improves next week?
- **Do not** use weekly review to brag about wins in community — process testimonials only.
- Review includes: gate pass rate, skip reasons, worksheet errors, sleep/distraction patterns.
- Set **one** improvement target for next week (e.g. "100% gate before demo entries").
- Schedule review same day/time each week — calendar block, non-negotiable.
- VIP desk expects you to arrive with review discipline — complete at least 2 weekly reviews before considering upgrade (M5).

## M3L3 — Setup documentation format

### Objective
Document trade ideas inside SOPs using the Alpha Elite structured format — education, not alerts.

### Video
`[PENDING: YouTube unlisted URL]` · Title: `AE Apprentice M3L3`

### Key points

- A **documented setup** includes: context, invalidation, risk %, size, trigger, and "what would cancel this idea."
- Structured trade ideas you see in VIP are **illustrations of format** — not instructions to copy blindly.
- Use neutral language: "if price does X, my plan was Y" — not "buy now."
- Store setups in journal or sheet linked from weekly review — build a personal library.
- Grade setups **after** outcome: process quality (A–F), not dollar result.
- Module 3 complete: 3 documented setups + 1 weekly review submitted to your own journal.
