# Automation Support Checklist

**Alpha Elite · Apprentice M4L2 · Education only**

Before relying on any bot, scanner, or copier — confirm manual ownership.

---

## Manual decisions (never outsource)

- [ ] I set max risk % per trade
- [ ] I set daily max loss limit
- [ ] I define when **not** to trade
- [ ] I run pre-trade risk gate myself
- [ ] I accept I am not buying passive income

## Tool deployment

- [ ] Tool tested on demo minimum _____ days
- [ ] Written SOP for what tool may / may not do
- [ ] Kill switch documented (how to disable fast)
- [ ] Broker permissions reviewed (no surprise leverage)

## Red flags (if yes, pause tool use)

- [ ] Tool places trades I did not plan
- [ ] Marketing promised "guaranteed" or "sleep income"
- [ ] I cannot explain stop logic for last 5 tool actions
