# Journal Field Guide

**Alpha Elite · Operating Toolkit · Education only**

Minimum fields for a learning-focused journal (not a P&L flex sheet).

---

| Field | Why log it |
|-------|------------|
| Date / session | Pattern detection over weeks |
| Instrument | Context for review |
| Setup ID | Links to setup documentation |
| Plan summary | Prove pre-meditation |
| Risk $ / % | Discipline tracking |
| Gate result (pass/skip) | Process KPI |
| Emotional vs structured | M1 metric |
| Outcome (optional) | Secondary to process |
| Lesson | One sentence improvement |

---

## Rules

- Log **skips** as proudly as trades — structure over frequency.
- Do not post journal screenshots as profit proof in Telegram.
- Weekly review pulls from these fields — see `weekly-review-template.md`.
