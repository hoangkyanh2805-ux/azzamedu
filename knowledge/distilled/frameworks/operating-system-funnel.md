# Framework — Operating System Funnel Pattern

## Pattern
```text
Framework (free) → Course (paid entry) → Membership (recurring) → Desk (ascension) → Service (DWY)
```

## Alpha Elite mapping
| Stage | Offer | System job |
|-------|-------|------------|
| Attract | Alpha Elite Gameplan (2% Rule inside) | Qualify + educate on risk |
| Activate | Apprentice Operating Course | Teach SOP execution |
| Retain | VIP Private System | Community + ongoing support |
| Ascend | Quant Desk / Inner Circle | Advanced layer |
| Implement | DWY Bot & Broker Setup | Done-with-you |

## Tech mapping
| Funnel stage | Tool |
|--------------|------|
| Attract | WP/Elementor + Brevo |
| Transact | WooCommerce + FunnelKit |
| Deliver | LearnHouse + YouTube Unlisted |
| Retain | Telegram + Brevo onboarding |
| Pay | PayPal + Crypto (manual MVP) |

## Reuse
Replace "2% rule" with any single mechanism your brand owns (e.g. content calendar, sales SOP, fitness periodization).
