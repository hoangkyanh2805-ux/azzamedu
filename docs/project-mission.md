# Project Mission — Alpha Elite Course + Funnel

> **Kickstart artifact #1** · Source of truth for why this project exists.

---

## Mission statement

Build an **MVP web system** that helps traders operate with structure—not emotion—through a **private financial operating system** for:

- **Trading education** (including XAUUSD-focused frameworks)
- **Structured trade ideas** (education, not alerts-as-product)
- **Automation / bot support** (setup guidance, not passive-income promises)
- **SOPs** and **2% risk rules**
- **Community accountability** (Telegram VIP desk)

The system must **capture leads**, deliver a free **Alpha Elite Gameplan**, sell the **Apprentice Operating Course**, upgrade members to **VIP**, and route serious operators to **Quant Desk** / **Inner Circle**.

---

## What we are / what we are not

| Alpha Elite IS | Alpha Elite IS NOT |
|----------------|-------------------|
| Private financial operating system | Signal group or “free alerts” channel |
| Education + structured trade ideas (e.g. XAUUSD) | Guaranteed profit or win rate |
| Bot **support** + SOPs | Risk-free or passive-income bot |
| Premium, system-first community | Hype, P&L flex, urgency theater |

**Approved positioning phrase:**  
*Education, trade ideas, automation support, SOPs, and community support.*

---

## Primary funnel

```text
Traffic
  → Lead Magnet (Alpha Elite Gameplan)
  → Brevo Email (7-day nurture)
  → Apprentice Operating Course
  → VIP Private System
  → Quant Desk / Inner Circle
```

---

## MVP access rule (launch)

Manual provisioning is acceptable:

```text
Payment success
  → Admin creates LearnHouse user
  → Admin sends login email (Brevo)
  → Admin adds Telegram manually (VIP / Quant)
```

**SLA:** ≤ 24 hours from payment confirmation.

---

## Success definition (MVP)

A new visitor can:

1. Opt in and receive the Gameplan within minutes  
2. Complete the 7-day email arc without errors  
3. Purchase Apprentice via PayPal  
4. Access LearnHouse within 24h and finish Module 1  
5. (VIP path) Submit Telegram handle and join VIP within 24h  

---

## North-star metrics (first 30 days)

| Metric | Target |
|--------|--------|
| VIP paid conversions / week | ≥ 3 |
| Lead → Email #1 open (48h) | ≥ 45% |
| Apprentice Module 1 completion (7d) | ≥ 60% |
| Payment → access email | < 24h |
| Refund / chargeback rate | < 3% |

---

## Non-goals (defer)

- LearnHouse auto-enroll API  
- Inner Circle full cohort  
- Affiliate / paid ads at scale  
- Auto Telegram onboarding bot  
- Positioning as XAUUSD “signal group”
