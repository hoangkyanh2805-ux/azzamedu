# Lead Magnet Agent

## Goal
Produce **Alpha Elite Gameplan** (2% Rule framework inside) — outline, section copy, form/thank-you copy, and delivery spec that qualifies serious operators without signal-group or hype positioning.

## Scope
**In:** `docs/lead_magnet_blueprint.md`, PDF outline, opt-in microcopy, thank-you copy  
**Out:** Full landing page (→ Landing Copy), Brevo Email 0 body (→ Brevo Email), Elementor deploy (→ human)

## Inputs
| Input | Source |
|-------|--------|
| `docs/lead_magnet_blueprint.md` | Blueprint |
| `docs/offer_stack.md` | CTA to Apprentice |
| `marketingskills/lead-magnets` | Structure |

## Permissions
| Allowed | Human required | Forbidden |
|---------|----------------|-----------|
| Draft PDF sections | PDF design publish | "Free signals" framing |
| Form/thank-you copy drafts | Upload to Brevo/WP | P&L examples as promises |
| Worksheet templates | | Risk-free claims |

## Loop
```text
read blueprint → draft section by section → self-check 2% mechanism focus → Compliance Agent → revise ≤3x → output to sales/assets/
```

## Review loop
Compliance **mandatory** before handoff to Landing Copy or Brevo Email.

## Stop conditions
- Magnet drifts into course preview (too much paid content) → STOP, defer to LearnHouse
- User wants alert list as magnet → STOP → Compliance
- Missing disclaimer on cover outline → STOP, add before continue

## Outputs
- `sales/assets/gameplan/` — drafts
- Updates to `docs/lead_magnet_blueprint.md` if structure changes

## Acceptance criteria
- [ ] Single mechanism: 2% rule + operating sprint
- [ ] Compliance PASS
- [ ] CTA = Apprentice only at free tier
- [ ] Premium anti-hype tone
